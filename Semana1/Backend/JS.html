<h1 id="javascript">JavaScript</h1>

<p>JavaScript es un lenguaje de programacion que puede ser aplicado a un documento HTML para agregarle interactividad, tambien se puede usar desde el lado del servidor con frameworks como Nodejs, ademas se puede usar para programar el microcontrolador de una placa Arduino usando la libreria Johnny-Five.</p>

<h2 id="variables">Variables</h2>

<ul> <li>En programacion las variables son contenedores para valores que pueden ser de diferentes tipos.</li> <li>Las variables tienen un nombre y un valor separados por un signo igual ‘=’.</li> <li>Las variables se declaran usando la palabra clave ‘var’ seguida del nombre de la variable.</li> <li>Los nombres de las variables pueden ser cualquier letra o palabra, pero existen algunas restricciones como por ejemplo, el nombre de una variable no puede comenzar por un numero, aunque si puede contener numeros en medio o al final del nombre de la variable, tampoco se puede usar ciertas palabras reservadas como nombre de una variable tales como <em>var</em>,<em>con</em>,<em>do</em>,etc.</li> </ul>

<p>En el primer ejemplo se definen dos variables con sus respectivos valores y se define una tercera variable con el valor de la suma de los anteriores.</p>

<h3 id="ejemplo-1">Ejemplo 1</h3>

<pre class="prettyprint"><code class="language-js hljs "><span class="hljs-keyword">var</span> x = <span class="hljs-number">20</span>, y = <span class="hljs-number">15</span>, suma = x + y;</code></pre>

<p>El interprete lee el codigo instruccion por instruccion, de modo que cada instruccion termina en un punto y coma.</p>

<p>En palabras sencillas el primer ejemplo realiza los siguientes pasos:</p>

<ul> <li>Define una nueva variable de nombre ‘x’ y le asigna un valor de ‘20’</li> <li>Define una nueva variable de nombre <em>y</em> y le asigna un valor de <em>15</em></li> <li>Define una nueva variable de nombre <em>suma</em> y le asigna el resulta de la suma de los valores de las variables <em>x</em> y <em>y</em></li> </ul>

<h3 id="tipos-de-variables">Tipos de variables</h3>

<h2 id="estructuras-condicionales">Estructuras condicionales</h2>

<p>Las condiciones son estructuras que te permiten verificar los valores de ciertas variables y dependiendo del resultado ejecutar un conjunto de instrucciones.</p>

<p>La estructura la condicional <code>if</code> es la mas facil y su estructura es <code>if (condicion) { instruccion;}</code>, de manera que si la condicion es verdadera el codigo encerrado entre llaves se ejecuta.</p>

<h3 id="ejemplo-3">Ejemplo 3</h3>

<pre class="prettyprint"><code class="language-js hljs "><span class="hljs-keyword">var</span> miEdad = <span class="hljs-number">21</span>; <span class="hljs-keyword">if</span> (miEdad >= <span class="hljs-number">18</span>){ alert(<span class="hljs-string">"Soy mayor de edad!"</span>); }</code></pre>

<p>En la mayoria de casos es necesario ejecutar otras instrucciones si el valor de la condicion <code>if</code> es falso eso se puede hacer usando la condicional <code>if else</code> y su estructura es <code>if (condicion) {instrucciones si es cierto} else { instrucciones si es falso}</code></p>

<h3 id="ejemplo-4">Ejemplo 4</h3>

<pre class="prettyprint"><code class="language-js hljs "><span class="hljs-keyword">var</span> miEdad = <span class="hljs-number">17</span>; <span class="hljs-keyword">if</span> (miEdad >= <span class="hljs-number">18</span>){ alert(<span class="hljs-string">"Soy mayor de edad!"</span>); } <span class="hljs-keyword">else</span> { alert(<span class="hljs-string">"Soy menor de edad"</span>) }</code></pre>

<p>Las condiciones tambien pueden ser anidadas de manera que se ejecuten si cumplen todas las condiciones previas.</p>

<h3 id="ejemplo-5">Ejemplo 5</h3>

<pre class="prettyprint"><code class="language-js hljs "><span class="hljs-keyword">if</span> (edad >= <span class="hljs-number">18</span> ){ alert(<span class="hljs-string">"Eres mayor de edad"</span>); <span class="hljs-keyword">if</span> (licencia == <span class="hljs-literal">true</span>){ alert(<span class="hljs-string">"Eres apto para conducir"</span>); } <span class="hljs-keyword">else</span> { alert(<span class="hljs-string">"Pero no puedes conducir"</span>); } } <span class="hljs-keyword">else</span> { alert(<span class="hljs-string">"Aun no puedes conducir por tu edad"</span>); }</code></pre>

<h2 id="bucles-o-loops">Bucles o loops</h2>

<p>Los bucles son condiciones repetitivas, donde una variable cambia de valor en cada ciclo. La forma más fácil de un bucle es usando la sentencia for. Ésta tiene una sintaxis que es similar a la sentencia if, pero con más opciones: <br> <code>for (valor inicial; condicion de fin; cambio)</code></p>

<h3 id="ejemplo-6">Ejemplo 6</h3>

<pre class="prettyprint"><code class="language-js hljs "><span class="hljs-keyword">for</span>(<span class="hljs-keyword">var</span> i = <span class="hljs-number">1</span>;i <= <span class="hljs-number">31</span>;i = i + <span class="hljs-number">1</span>){ alert(<span class="hljs-string">"Hoy es el dia "</span> + i + <span class="hljs-string">" del mes!"</span>) }</code></pre>

<p>Una aplicacion importante de los bucles es para recorrer los valores de un array.</p>

<h3 id="ejemplo-7">Ejemplo 7</h3>

<pre class="prettyprint"><code class="language-js hljs "><span class="hljs-keyword">var</span> cursos = [<span class="hljs-string">'Backend'</span>,<span class="hljs-string">'Frontend'</span>, <span class="hljs-string">'Electronica'</span>] <span class="hljs-keyword">var</span> total = cursos.length; <span class="hljs-keyword">for</span>(<span class="hljs-keyword">var</span> i=<span class="hljs-number">0</span>;i<total; i++){ alert(<span class="hljs-string">"Estoy tomando el curso de "</span> + cursos[i] + <span class="hljs-string">" en el CoreUpgrade 2015"</span>) }</code></pre>
